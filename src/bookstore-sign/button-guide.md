# Bookstore Sign - Quick Button Guide

## Setting the mode

The sign has two buttons:

- **RIGHT button (BUTTON_1)** → go to the **NEXT** mode
- **LEFT button (BUTTON_2)** → go to the **PREVIOUS** mode

Press a button once to change modes. The on-board LED blinks each time the mode
changes. The modes wrap around - pressing **NEXT** past the last mode returns to
Mode 0.

## The modes

| Mode | Name | What it does |
|:----:|------|--------------|
| **0** | Auto-Cycle | Plays every pattern by itself, 60 seconds each |
| **1** | Color Letters | Each letter of BOOKSTORE glows its own color |
| **2** | Rainbow Flow | A rainbow scrolls across the sign |
| **3** | Rainbow Solid | The whole sign glows one slowly-changing color |
| **4** | Color Sparkle | Pixels shimmer through random colors |
| **5** | Color Wipe | The sign changes to a new bright color every second |
| **6** | Comet | A bright dot races across with a glowing tail |

**Tip:** Start in Mode 0 to watch all the patterns, then use the buttons to stop
on your favorite one.
