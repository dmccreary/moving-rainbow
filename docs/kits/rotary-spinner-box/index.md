# Rotary Spinner Project Documentation

## Kit Description

A small clear plastic box with a 12-pixel LED strip inside
that is controlled by a rotary encoder and two momentary push buttons.

The kit includes the following components.

- Raspberry Pi Pico microcontroller ($4-6 - add $1 for using the Pico-H with pre-soldering headers)
- Rotary encoder with built-in push button
- 1/2-size breadboard with 400 tie points
- 12-pixel addressable NeoPixel RGB strip (WS2812B) (not a ring)
- One extra momentary push buttons
- Power switch
- 3 AA batteries and battery case
- 22 gauge solid jumper wires
- USB cable(s) - some kits include both a USB-A and USB-C cable
