# Cover Image Prompt

!!! prompt
from the /book-installer run the guide that will generate the image prompt for the cover image.  Put the welcome mascot image in the lower left corner and place the title "Moving Rainbow" in the center using a colorful color-changing font color.  Behind the title place a montage of images from the class: an LED Strip with a rainbow pattern, the Raspberry Pi Pico on a breadboard, a checklist of the things we learn in computational thinking (decomposition, abstraction, patterns etc) and images of kids with LED costumes having fun!
